#How Users Install & Run

  Step 1: Download & Extract

  # Users download DoctypeEngine_1.0.0.zip
  # Double-click to extract

  Step 2: Run (First Time)

  cd DoctypeEngine
  ./start.sh

  First run does:
  - Creates virtual environment automatically
  - Installs all dependencies
  - Sets up database
  - Starts server
  - Opens on http://127.0.0.1:8000

  Takes 2-3 minutes first time.

  Step 3: Use

  - Browser opens automatically
  - No terminal knowledge needed
  - Works offline after setup

  #Why This is Better

  Simple Package Advantages:
  - No Python runtime bundling needed
  - macOS already has Python
  - Users can see and trust the code
  - Easy debugging if issues arise
  - Can customize settings easily
  - Updates are just file replacements
  - Works on both macOS and Linux

  DMG/PyInstaller Issues:
  - Complex build process
  - Large file size (100x bigger)
  - Build can fail with updates
  - Hard to debug issues
  - Can't easily customize
  - macOS Gatekeeper warnings
  - Requires code signing ($99/year)
